jQuery.deferred()参考资料

deferred.done( doneCallbacks [, doneCallbacks ] )
:添加functions，当deferred被解决时候触发

deferred.fail( doneCallbacks [, doneCallbacks ] )
:添加functions，当deferred被拒绝时候触发

deferred.then(resolvedCallback[, rejectedCallback [, progressCallback]])
:resolvedCallback 当deferred被解决时候触发
:rejectedCallback 当deferred被拒绝时候触发 :progressCallback 当deferred 在处理过程中候触发 :return :A Promise object

deferred.always(callbacks[, callbacks, ..., callbacks])
:无论 deferred是被解决或者拒绝，callbacks都会被执行

deferred.state()
:返回一个表示deferred对象的state

....

then(onFulfilled, onRejected, onProgress):on方法会被异步执行(setTimeout(...))
